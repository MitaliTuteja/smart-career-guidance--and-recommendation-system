# 🚀 Smart Career Guidance and Recommendation System

An AI-powered web application that provides personalized career recommendations based on a student's academic background, skills, personality, interests, and work style. The system uses Machine Learning to help students make informed career decisions and identify skill gaps.

🌐 **Live Demo:** https://career-guide-ai-mitali.onrender.com
*(Replace this with your actual Render URL if different.)*

---

## 📌 Project Overview

Choosing the right career path can be challenging for students due to the availability of numerous career options. This project addresses this problem by leveraging Machine Learning to recommend suitable career paths based on user inputs.

The system analyzes multiple factors such as academic score, technical skills, soft skills, personality, interests, and work style to generate accurate career recommendations.
